# Line Plotter
Plots a linear function.

This supports CLI arguments. For example: `./a.out -2 2` will plot a line with a gradient of -2 and a y-int of 2.
## Usage
```
// Compile the code, GCC is used as default but you can use whatever compiler you prefer
gcc line_plotter.c

// Assuming a.out is the compiled binary
./a.out
```
